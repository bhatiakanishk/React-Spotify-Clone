## Running the app
a) Clone the repository
```
git clone https://github.com/bhatiakanishk/React-Spotify-Clone
```

b) Go to cloned repository
```
cd spotify-clone-app
```

c) Run the app
```
npm start
```

## Technologies implemented
a) User Authentication

b) Spotify API

c) Responsive Design

d) Material UI

e) React Context API

f) React Hooks
